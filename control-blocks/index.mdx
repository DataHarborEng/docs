# Govern Data

This section covers the part of the pipeline where DataHarbor changes what the response contains and who can access it.

At a high level:

1. Upstream data is normalized into canonical JSON.
2. The matching object definition is selected.
3. The ordered controls pipeline runs.
4. The governed result is formatted for delivery.

Use the pages below to decide which fields to protect, how to transform them, and how field paths behave across nested objects and arrays.
