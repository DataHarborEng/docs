# Build Virtual APIs

DataHarbor lets you turn one enrolled source into multiple governed views without cloning the data.

This section is organized around the lifecycle of a Virtual API:

1. Understand the Virtual API model.
2. Learn the request pipeline from normalization through formatting.
3. Enroll a source and connect access paths.
4. Decide how people, systems, and agents will consume the result.
